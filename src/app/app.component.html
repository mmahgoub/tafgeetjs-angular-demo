TafgeetJS

Convert currency digits into written Arabic words

{{amount | currency:'SDG':true:'1.2-2'}}

{{title}}

How to use TafgeetJS with Angular 4

First install the npm module using $npm install tafgeetjs --save Then it is a matter of importing tafgeetjs into your component like so:

import * as Tafgeet from 'tafgeetjs';

let amount = 1000.20;
let amountInText = new Tafgeet(amount, 'USD').parse();
